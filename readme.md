#  ձայնակոյտ

- այսէ հաշուարկ հայերէն բառերում առկայ ուրոյն ձայնակոյտերի վիճակագրութեան
- աւգտագործուել են 1895 թուականի գրաբար աստուածաշունչն ու 1836 թուականի հայկազեան բառարանը
- ձայնաւորների regex աւրինաչափութեամբ - `(?:((?<![աէոօեիը])ւ)|(?:[աէոօեիը]+վ(?![աէոօեիը]|$)|[աէոօեիը]+[ւյ]?))`
- բաղաձայնների regex աւրինաչափութեամբ - `[^\n\sաէոօեիըւ0-9]+`
- արդիւնքները գրուել են համապատասխան csv ֆայլերում
