#  ձայնակոյտ

այսէ հաշուարկ հայերէն բառերում առկա ուրոյն ձայնակոյտերի վիճակագրութեան։ աւգտագործուել են 1895 թուականի գրաբար աստուածաշունչն ու 1836 թուականի հայկազեան բառարանը։

հաշուարկն անցել է իւրաքանչիւր բառացանկի վրայով հետեւեալ regex աւրինաչափութեամբ - `(?:((?<![աէոօեիը])ւ)|(?:[աէոօեիը]+վ(?![աէոօեիը]|$)|[աէոօեիը]+[ւյ]?))`։ արդիւնքները գրուել են համապատասխան csv ֆայլերում։
